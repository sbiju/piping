Welcome to the new erp application to ease data entry jobs in plant piping. This cloud erp system is made exclusively to improve data handlling and document processing during construction phase of process plant piping, this enables to connect different departments and summarize the reports in effective way and save time in documentation. Different depattment staff can perform their jobs and view reports and status.
Main features of the application are:
  • Effective and timely co-ordination of different departments
  • Enable to enter and view the various reliorts status easily without delay and errors
  • Data will be saved in cloud and backed up to prevent loss of data due to system malfunctions and errors
  • Data is being transmitted in encrypted form so that others cannot read or copy
  • Main data entry takes place only in one place and others update from related departments, so that there will not be any double work
  • No need to send reports to any persons as they can view reports online at the time when it updates
  • System calculated material summary and performance of resources makes effective monitoring
  • Can be saved in excel and pdf formats
  • Easy to implement and use
  • Permitted customization will be added on request
  • This is a modern python based erp system with built in security so it is trust worthy in current scenario
  • Take a free trial for one month and experience the features
  • Individual applications are also available on request

Start up Instructions

  • Click on the Site Admin Link and Create seven users, and assign one department to each user
  • Distribute username and password to concerned users.
  • Add a project with project name using the add project link.
  • Initially data entry user can login to the system and start entering datas
  • Each department users can login and update the reports on relevent fields on daily basis
  • Construction heads and clients can view the reports
  • Whenever necessary to remove a user, delete and create a new user and assign to the same department where the user is deleted
  • Instructiions for each users can be seen on their respective pages.
